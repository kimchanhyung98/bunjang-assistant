# Claude Skill

번개장터 판매글 작성을 돕는 Claude용 스킬입니다.

## 목적

- 상품 이미지를 분석해 상품 유형과 상태를 식별합니다.
- 유사 판매글을 바탕으로 시세 조사 포인트를 정리합니다.
- 대표 이미지(썸네일) 후보를 우선순위와 함께 추천합니다.
- 판매글 설명 초안과 번개장터 입력 항목 초안을 생성합니다.

## 입력

- 상품 이미지 여러 장
- 상품명 또는 알고 있는 제품 정보
- 구성품, 하자, 사용감, 구매 시기 등 판매자가 알고 있는 정보
- 시세 조사를 위해 참고할 키워드 또는 브랜드/모델명

## 작업 순서

1. **이미지 분석**
   - 상품 카테고리, 브랜드, 모델명, 색상, 재질, 크기 단서를 추출합니다.
   - 오염, 찍힘, 변색, 사용감 등 상태 정보를 식별합니다.
   - 구성품 유무와 촬영 품질을 함께 정리합니다.
2. **시세 조사 지원**
   - 검색에 사용할 핵심 키워드 조합을 제안합니다.
   - 유사 판매글 비교 시 확인해야 할 항목(상태, 구성품, 거래 방식)을 정리합니다.
   - 조사 결과를 입력받으면 적정 가격대 범위를 요약합니다.
3. **대표 이미지 추천**
   - 상품이 가장 선명하고 정면에 가깝게 보이는 이미지를 우선 추천합니다.
   - 오염/하자가 보이는 사진은 상세 설명용으로 분리합니다.
   - 추천 사유를 한 줄씩 제공합니다.
4. **판매글 초안 작성**
   - 번개장터 제목 초안
   - 핵심 특징 중심의 설명 초안
   - 상태/하자/구성품/거래 조건 요약
   - 번개장터 입력 항목용 요약값

## 작성 원칙

- 이미지에서 확실히 보이지 않는 정보는 추정이라고 명시합니다.
- 하자나 사용감은 축소하지 않고 판매글에 반영합니다.
- 과장 표현보다 검색성과 신뢰도를 우선합니다.
