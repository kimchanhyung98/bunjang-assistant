# 검색 결과 Fixture

이 문서는 실제 번개장터 조회 없이 구조화된 검색 결과를 요약하는 방식을 확인할 때만 읽습니다.

## 고정 데이터

- `fixtures/search-result.json`: 검색 결과 JSON 예시

## 사용 시점

- 스킬 동작을 검증할 때
- `search.listings` 또는 `agent-search-rank` 결과를 사람이 읽기 쉬운 후보 요약으로 바꾸는 기준을 확인할 때
- 실제 계정, 실제 매물, 실제 번개장터 상태를 건드리지 않고 응답 형태를 점검할 때

## 요약 기준

- 각 후보의 `listingId`, `title`, `price`, `condition`을 빠뜨리지 않습니다.
- 가격은 원 단위로 읽기 쉽게 표시합니다.
- fixture 데이터에 없는 하자, 구성품, 판매자 정보, 거래 가능 여부는 추정하지 않습니다.
- 후보가 여러 개이면 비교 가능한 차이점부터 요약합니다.
