# Codex Skill

번개장터 판매글 작성을 돕는 Codex용 스킬입니다.

## 목적

- 상품 이미지를 분석해 핵심 특징을 추출합니다.
- 시세 조사 결과를 바탕으로 판매가 제안을 지원합니다.
- 대표 이미지(상품 썸네일) 선정을 지원합니다.
- 번개장터 판매글 설명과 입력 자동화 초안을 생성합니다.

## 입력

- 상품 이미지 여러 장
- 판매자가 알고 있는 상품 정보
- 시세 조사 결과(유사 상품 가격, 판매 완료 여부, 상태 차이)
- 거래 조건(배송 가능 여부, 직거래 지역 등)

## 작업 순서

1. **핵심 특징 추출**
   - 상품 종류, 브랜드/모델, 색상, 재질, 크기, 눈에 띄는 디자인 요소를 정리합니다.
   - 사용 흔적, 하자, 구성품 포함 여부를 구조적으로 정리합니다.
2. **판매가 제안**
   - 시세 조사 결과를 기준으로 최저가/권장가/희망가 범위를 제안합니다.
   - 가격 산정 근거를 상태, 구성품, 희소성 기준으로 설명합니다.
3. **썸네일 선정**
   - 대표 이미지 후보를 순위별로 제안합니다.
   - 선명도, 상품 가시성, 배경 방해 요소 여부를 기준으로 판단합니다.
4. **판매글/입력 자동화**
   - 검색 친화적인 제목을 작성합니다.
   - 신뢰도 있는 설명 문단과 불릿 요약을 생성합니다.
   - 번개장터 입력칸에 바로 붙여넣을 수 있는 요약값을 제공합니다.

## 출력 형식

```md
## 핵심 특징
- 상품 유형:
- 브랜드/모델:
- 눈에 띄는 특징:
- 상태/하자:
- 구성품:

## 판매가 제안
- 최저가:
- 권장가:
- 희망가:
- 산정 근거:

## 썸네일 추천
1. 이미지 n - 추천 이유
2. 이미지 n - 추천 이유

## 판매글 초안
- 제목:
- 설명:

## 입력 자동화용 필드
- 카테고리:
- 가격:
- 상태:
- 구성품:
- 거래 방법:
```

## 작성 원칙

- 확인되지 않은 정보는 단정하지 않습니다.
- 시세 조사 결과가 부족하면 추가 조사 키워드를 먼저 제안합니다.
- 판매가 제안은 근거와 함께 제공해 사용자가 쉽게 조정할 수 있게 합니다.
