# 스킬 설치와 사용

기준 스킬은 다음 파일입니다.

```text
skills/bunjang/SKILL.md
```

스킬 직접 설치:

```bash
./install/install-skills.sh --tool codex --scope project
./install/install-skills.sh --tool claude --scope project
```

사용자 범위 설치:

```bash
./install/install-skills.sh --tool codex --scope user
./install/install-skills.sh --tool claude --scope user
```

공개 스킬의 세부 흐름은 `skills/bunjang/references/`에 둡니다.
